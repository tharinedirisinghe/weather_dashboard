# Screenshots Directory

This directory contains screenshots for the Weather Dashboard app documentation.

## Required Screenshots

1. **online_mode.png** - App showing live weather data
2. **offline_mode.png** - App displaying cached data with "(cached)" tag
3. **error_handling.png** - App showing error message

## How to Capture Screenshots

### For Android Emulator:
- Use the camera icon in the emulator toolbar
- Or press `Ctrl + S` (Windows) / `Cmd + S` (Mac)

### For Physical Device:
- Use device's screenshot functionality
- Transfer images to this folder

## Naming Convention
Keep the filenames as specified above for proper README display.
